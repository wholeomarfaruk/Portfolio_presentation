our journey

2017 - 2018

Basic Computer

The computer's operating system manages the computer's resources, such as memory and input/output devices. Applications or software programs are installed on the computer's operating system, and are used to perform specific tasks or functions, such as word processing or browsing the internet.

2018 - 2019

Basic HTML & CSS

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a web page. CSS allows web developers to control the layout, typography, and colors of a web page.

2019 - 2020

One page website

A basic one page website is a website that consists of only one web page that contains all the necessary information about a business, product, or service. It is a simple and effective way to establish an online presence and communicate your message to potential customers.

2020 - 2021

Multiple website

A multiple website, also known as a multi-site, is a type of website that allows for the creation and management of multiple websites from a single platform or content management system (CMS). This allows organizations to have multiple websites that are related to each other or serve different purposes, all under one umbrella.

2021 - 2022

Dynamic website design

Dynamic websites can have interactive features such as login systems, forms, search bars, shopping carts, and personalized content. They often use a database to store and retrieve data, allowing for dynamic content to be generated on-the-fly

2022 - 2023

Backend development

Backend development involves building the server-side of web or mobile applications, which includes designing and implementing the logic, database management, and APIs. It typically involves programming in languages like Python, Java, or Node.js.
